# 框架

# vue

官方文档

  • element - 饿了么出品的 web 库

  • mint - 饿了么出品的移动端库

  • cube - 滴滴出品的移动端库，感觉比 mint 完善一点

# react

官方文档（中文版）

  • antd - 阿里 react 框架，星星很多哦

  • antd-pro - antd 阿里官方解决方案，拉了代码就可以用，非常方便

  • react-admin - 网友 react+antd 写的系统模板，很多人在没出 antd-pro 时都用这个

# 小程序

  • mpvue - 轻松使用 vue 敲代码，编译成小程序

  • 小程序开发文档

  • wxParse - 在小程序中解析 html 和 markdown

  • mpvue-wxParse - 在使用 mpvue 搭建小程序中解析 html 和 markdown，star 比较少，大家用的如果好的话可以给作者一个 star，鼓励一下作者
